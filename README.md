# Hard&Custom - E-Commerce de Hardware y Periféricos

## Temática y Público Objetivo
Nuestro e-commerce se especializa en la venta de hardware de alto rendimiento y periféricos customizables. Ofrecemos desde componentes clave para la optimización de equipos (módulos de memoria RAM DDR5, unidades SSD, monitores de alta tasa de refresco) hasta teclados mecánicos para la personalizacion de equipos

## Descripción Personal
Soy Agustín Cárdenas, estudiante de cuarto año de Ingeniería en Informática. Me apasiona el desarrollo backend, la arquitectura de sistemas y la optimización de rendimiento a nivel de hardware y software. Mi enfoque está en crear soluciones robustas y escalables utilizando tecnologías como Node.js y bases de datos relacionales, aplicando buenas prácticas de diseño y lógica de negocio.

## Sitios de Referencia
1. **CompraGamer:** Elegido por la estructura de su catálogo de componentes de hardware y la claridad de su carrito de compras paso a paso.
2. **Redragon (Sitio Oficial):** Seleccionado por la estética visual en la presentación de periféricos inalámbricos y teclados mecánicos.
3. **Keychron:** Tomado como referencia por la excelente experiencia de usuario (UX) al permitir filtrar teclados por tipo de switch, formato y conectividad.
4. **Crucial:** Referencia clave para la sección de "Detalle de producto", especialmente en cómo muestran las especificaciones técnicas y compatibilidad de memorias RAM y almacenamiento.
5. **Mercado Libre:** Elegido por la funcionalidad y usabilidad de sus formularios de registro y login, siendo un estándar en la región para transacciones digitales.
